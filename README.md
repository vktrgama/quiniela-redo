# Quiniela WorldCup 2022 Qatar

This project is redo React version of my [Quiniela for WorldCup 2018](https://github.com/vktrgama/quiniela).

### Design and Architecture

-   React 18.0
-   Material UI 5.8.5
-   AWS Amplify Back-end services:
    -   Authentication
    -   API
    -   GraphQL
    -   DynamoDb
